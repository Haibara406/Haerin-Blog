# 🌟 二分查找的本质

二分查找本质只有一句话：

> 在一个具有“单调性”的区间里，找边界。

所有题型都可以归类为：

1. 找某个值
2. 找左边界（第一个满足）
3. 找右边界（最后一个满足）
4. 在答案空间里二分（不是在数组上）

------

# 🧩 一、普通查找（是否存在）

适用场景：

- 查找 target 是否存在
- 返回任意一个 target 位置

### ✅ 模板

```java
while (left <= right) {
    int mid = left + (right - left) / 2;

    if (nums[mid] == target) {
        return mid;
    } else if (nums[mid] < target) {
        left = mid + 1;
    } else {
        right = mid - 1;
    }
}
return -1;
```

### 特点

- `while (left <= right)`
- 找到就 return
- 典型“教科书二分”

------

# 🧩 二、找左边界（第一个 >= target）

适用场景：

- 第一个等于 target
- 第一个 >= target
- lower_bound
- 查找插入位置

------

## ✅ 模板（左闭右闭）

```java
while (left < right) {
    int mid = left + (right - left) / 2;

    if (nums[mid] >= target) {
        right = mid;
    } else {
        left = mid + 1;
    }
}
return left;
```

### 记忆口诀：

> 找左边界
> mid 向下取整
> 满足 → 收右边

------

# 🧩 三、找右边界（最后一个 <= target）

适用场景：

- 最后一个等于 target
- 最后一个 <= target
- upper_bound - 1

------

## ✅ 模板

```java
while (left < right) {
    int mid = left + (right - left + 1) / 2;

    if (nums[mid] <= target) {
        left = mid;
    } else {
        right = mid - 1;
    }
}
return left;
```

### 记忆口诀：

> 找右边界
> mid 向上取整
> 满足 → 收左边

------

# 🧩 四、旋转数组找最小值

适用场景：

- 无重复旋转数组（153）

------

## ✅ 模板

```java
while (left < right) {
    int mid = left + (right - left) / 2;

    if (nums[mid] > nums[right]) {
        left = mid + 1;
    } else {
        right = mid;
    }
}
return nums[left];
```

### 核心思想：

> 用右端点当锚点
> 找断点右边第一个元素

------

# 🧩 五、有重复的旋转数组（154）

```java
while (left < right) {
    int mid = left + (right - left) / 2;

    if (nums[mid] > nums[right]) {
        left = mid + 1;
    } else if (nums[mid] < nums[right]) {
        right = mid;
    } else {
        right--;
    }
}
return nums[left];
```

------

# 🧩 六、答案空间二分（最重要）

适用场景：

- 最小可行值
- 最大可行值
- 木材切割
- 吃香蕉速度
- 最小时间
- 最小容量

本质：

> 在“答案区间”上二分，而不是在数组上。

------

## ✅ 模板（找最小可行解）

```java
while (left < right) {
    int mid = left + (right - left) / 2;

    if (check(mid)) {
        right = mid;
    } else {
        left = mid + 1;
    }
}
return left;
```

------

## ✅ 模板（找最大可行解）

```java
while (left < right) {
    int mid = left + (right - left + 1) / 2;

    if (check(mid)) {
        left = mid;
    } else {
        right = mid - 1;
    }
}
return left;
```

------

# 🔥 二分模板总口诀

## ① 找左边界

```
mid = (l+r)/2
满足 → r = mid
不满足 → l = mid+1
```

## ② 找右边界

```
mid = (l+r+1)/2
满足 → l = mid
不满足 → r = mid-1
```

------

# 🌟 二分的真正理解

所有二分题，本质都是：

> 在一个“单调函数”中找临界点

单调函数可能是：

- 数组本身单调
- 旋转数组的某种特性
- check(mid) 单调
- 时间、容量、速度等单调

------

# 🚀 终极总结

真正要记住的只有两种：

| 类型     | mid 取法 | 满足时 |
| -------- | -------- | ------ |
| 找左边界 | 下取整   | 收右边 |
| 找右边界 | 上取整   | 收左边 |

其余全部是变形。
